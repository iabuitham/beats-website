Beats Events Production — Full Rewrite (Static)

How to run:
- Open index.html in a browser, or
- Use VS Code "Live Server" on this folder (recommended).

Structure:
- assets/css/styles.css
- assets/js/main.js
- assets/img/placeholders/logo.svg

Notes:
- This rewrite does NOT require real images to look good (uses gradient placeholders).
- To add real images, replace the gradient classes (bg1..bg8) with <img> tags or CSS background-image.
- Contact form composes an email via mailto (no backend).
